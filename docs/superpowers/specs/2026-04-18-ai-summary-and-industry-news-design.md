# AI 요약 + 건설산업 뉴스 확장 설계

**작성일:** 2026-04-18  
**상태:** 승인됨

---

## 개요

기존 공제조합 이슈 모니터 시스템에 두 가지 기능을 추가한다:

1. **건설산업 키워드 확장** — 임직원이 관심 가져야 할 건설산업 뉴스(법·제도, 시장·경기, 안전·노동) 키워드 추가
2. **Claude AI 개별 요약** — 수집된 각 기사를 Claude API로 요약하여 articles.json에 저장, 이메일과 웹사이트에 모두 표시

---

## 아키텍처

기존 흐름에 summarizer 단계 추가:

```
crawler → 새 기사 목록
        → summarizer (Claude API, 기사별 요약)
        → article_store (articles.json에 summary 포함 저장)
        → mailer (이메일에 summary 포함)
        → index.html (웹사이트 카드에 summary 표시)
```

## 건설산업 키워드

기존 4개 공제조합 키워드 외 추가:

| 카테고리 | 키워드 |
|----------|--------|
| 법·제도 | `건설산업기본법`, `국토교통부 건설`, `건설업 규제` |
| 시장·경기 | `건설경기`, `건설 PF`, `건설수주` |
| 안전·노동 | `중대재해 건설`, `건설현장 안전`, `건설 노동` |

웹사이트에서는 건설산업 키워드 기사를 "건설산업" 필터 버튼 하나로 묶어 표시.

## 데이터 구조

### articles.json 기사 스키마

```json
{
  "keyword": "건설공제조합",
  "title": "기사 제목",
  "link": "https://...",
  "description": "원문 요약...",
  "summary": "Claude AI가 생성한 2~3줄 요약",
  "collected_at": "2026-04-18T10:00:00"
}
```

- `summary` 필드: Claude API 호출 성공 시 채워짐, 실패 시 `null`로 저장 (기존 description 표시로 fallback)

---

## summarizer.py 설계

```python
# 기사 하나씩 Claude claude-haiku-4-5-20251001 호출
# 입력: title + description
# 출력: 2~3줄 한국어 요약
# 실패 시: None 반환 (main.py에서 None이면 summary 필드 생략)
```

- 모델: `claude-haiku-4-5-20251001` (속도·비용 최적)
- 비용: 기사당 약 $0.0005 수준
- 실패 처리: 네트워크/API 오류 시 None 반환, 로그 기록, 전체 흐름 중단 없음
